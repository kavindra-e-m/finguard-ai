Quick Start (3 options):

📌 OPTION 1: Using Docker Compose (Recommended)
   ─────────────────────────────────────────────
   docker-compose up --build
   
   Then:
   1. Train models: docker exec finguard-ml-service python ml/train_all_models.py
   2. Seed data: docker exec finguard-ml-service python data/seed_data.py
   3. Open browser: http://localhost:5173


📌 OPTION 2: Local Development (Windows)
   ──────────────────────────────────────
   1. Start PostgreSQL (ensure running on localhost:5432)
   2. cd ml-service && python -m uvicorn main:app --reload --port 8000
   3. cd backend && mvnw.cmd spring-boot:run
   4. cd frontend && npm run dev
   5. Open browser: http://localhost:5173


📌 OPTION 3: Auto-Start Script (Windows)
   ────────────────────────────────────────
   start-local.bat
   
   (Opens services in separate windows)

Issue: Services won't start
  → See LOCAL_DEVELOPMENT.md → Troubleshooting section

Issue: Can't connect to ML Service
  → Check ML_SERVICE_URL in backend/src/main/resources/application.properties
  → Verify http://localhost:8000/health returns healthy status

Issue: CORS errors
  → Update cors.allowed-origins in application.properties
  → Must include http://localhost:5173

Issue: Database connection fails
  → Ensure PostgreSQL is running on localhost:5432
  → Verify credentials: user=finguard_user, pass=finguard_pass

Issue: JWT token errors
  → Clear browser localStorage and re-login
